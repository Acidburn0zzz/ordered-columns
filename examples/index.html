Getting Started

Setup

  1. Add the library using the Javascript package manager Bower: bower install --save ordered-columns
  2. Import Web Components' polyfill: <script src="bower_components/platform/platform.js"></script>
  3. Import Custom Element: <link rel="import" href="bower_components/ordered-columns/ordered-columns.html">

  4. Start using it!
